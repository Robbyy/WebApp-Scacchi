# Pezzi scacchi - set "cburnett"

I file SVG dei pezzi in questa cartella appartengono al set **cburnett**,
creato da Colin M. L. Burnett.

- Stile: Staunton classico.
- Origine: insieme di pezzi storicamente usato da Wikipedia e Lichess.
- Licenza: multi-licenza GFDL / BSD / GPL (vedi pagina dell'autore su Wikimedia
  Commons). Uso consentito con attribuzione.

Mappa nomi file: `<colore><Pezzo>.svg` dove colore = `w` (bianco) / `b` (nero)
e Pezzo = `K Q R B N P`. Esempio: `wK.svg` = re bianco, `bQ.svg` = donna nera.
